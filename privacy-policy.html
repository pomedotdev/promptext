Privacy Policy

Last updated: May 17, 2026

What Promptext collects

Promptext collects anonymous usage analytics to understand how the extension is used and to improve it over time. No personal information is ever collected.

Anonymous identifier

On first use, Promptext generates a random UUID and stores it in the extension's local storage. This identifier is not linked to your identity, browser account, or any other personal data. It exists solely to deduplicate events (e.g. to distinguish "10 users each copied XML once" from "1 user copied XML 10 times").

Usage events

The following events are sent to PostHog, a third-party analytics service:

Event When it fires
prompt_createdA new prompt is saved for the first time
prompt_deletedA prompt is deleted
prompt_duplicatedA prompt is duplicated
prompt_xml_copiedThe prompt XML is copied to the clipboard
prompt_opened_in_claudeA prompt is sent to Claude.ai
prompt_opened_in_chatgptA prompt is sent to ChatGPT
snippet_added_to_galleryA prompt element is saved to the snippet gallery
snippet_deletedA snippet is deleted
snippet_saved_from_context_menuA text selection is saved as a snippet via right-click
s3_sync_configuredS3 sync settings are saved
s3_sync_manualA manual S3 sync is triggered
s3_connection_testedThe S3 connection is tested from settings

Each event carries only the anonymous identifier described above and, where noted, a small set of non-personal properties (e.g. whether a custom S3 endpoint is used, or the XML tag name of a saved snippet).

No prompt content, no S3 credentials, no URLs, and no browser or system identifiers are ever sent.

What Promptext does not collect

  • The content of your prompts or snippets
  • Your S3 credentials, bucket name, or any sync data
  • Your IP address (PostHog is configured to not store it)
  • Your name, email, or any account information
  • Browsing history or data from pages you visit

Where data goes

Events are sent to PostHog and stored on their infrastructure. PostHog's own privacy policy is available at posthog.com/privacy.

Your data, locally

All prompts, snippets, and settings are stored exclusively in your browser's local extension storage (chrome.storage.local). If you configure S3 sync, that data is also stored in your own S3 bucket — Promptext has no access to it.

Changes to this policy

If the set of collected events changes materially, this document will be updated with a new date at the top.
